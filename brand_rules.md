# DONNA Brand Knowledge Base (Source of Truth)

This document provides persistent, explicit instructions derived from our codebase and brand guides. Use this before EVERY visual or copy generation.

## 1. Color Palette

Map of code design tokens (CSS variables) to descriptive natural language for image engines:

*   **Pure Black Background** (`oklch(0 0 0)`): A deep, ink-black, infinite space background. Use as the primary surface.
*   **Warm White Foreground** (`oklch(0.96 0 0)`): A soft, luminous off-white. Safe for primary text and clean UI elements.
*   **Futuristic Violet** (`oklch(0.65 0.23 308)`): Premium, high-energy royal violet. Used as a secondary glow or accent.
*   **Tech Mint** (`oklch(0.95 0.1 276)`): Cool, energetic mint/cyan-green. Represents speed, efficiency, and logic.
*   **Electric Cyan** (`oklch(0.85 0.18 225)`): Vivid electric cyan blue. The main brand accent. Highly vibrant neon highlights.
*   **Muted Space Grey** (`oklch(0.25 0 0)`): Subtle, clean grey border or background shades.

## 2. Typography

*   **Safe Font Families**: Modern, clean, geometric sans-serif (Geist Sans, Inter, Roboto, Apple San Francisco).
*   **Styling**: Thin-to-bold contrast, high tracking/letter-spacing for headings, monospaced accents for data/metrics.
*   **Copy Guidelines**: Avoid overly cursive or highly stylized display fonts that decay into visual noise. All rendered text must be crisp and perfectly horizontal.

## 3. Visual Style & Aesthetic

*   **Theme**: iOS-style Liquid Glassmorphism, refractive glass borders, subtle background blurs, neural stacked architecture, premium space-age futurism.
*   **Prohibitions**: 
    *   NO cartoon robots.
    *   NO human-like AI avatars or faces representing "AI."
    *   NO clunky circuits or generic "cyber" aesthetics.
*   **Aesthetic Vibe**: Highly professional, technical, infrastructure-grade, clean, and calm.

## 4. Voice & Messaging Tone

Any text written as overlay or copy must adhere to these parameters:

*   **Core Identity**: "Operational Intelligence Layer," "AI-assisted operations," "Workflow Coordination Infrastructure."
*   **Tone**: Calm, Intelligent, Business-focused, Helpful, Professional.
*   **Tone to Avoid**: Overhyped tech jargon, sentient "AI employee" framing, magic/unlimited promises.
*   **Approved Taglines**: 
    *   "Operational Intelligence for Modern Business."
    *   "Streamline Your Systems."
    *   "Calm Operations. Scalable Workflows."
